# CraftOrdersHelper

Tracks your favorited Crafting Order recipes and shows exactly which reagents you need to buy when you open the Auction House.

## Features

- **Auto-capture** — favorite any recipe in Crafting Orders, the addon remembers the reagents
- **AH panel** — shows automatically when you open the Auction House
- **Quality tracking** — counts low and high quality reagents separately (gray star / gold star)
- **One-click search** — click any reagent to instantly search it in the AH
- **Bag counts** — shows Need / Have / Buy for every reagent
- **Filters** — switch between This Character / Account, and All Sources / Craft Orders / Professions

## How to use

1. Favorite a recipe in **Crafting Orders** (click the ⭐ star)
2. Open the **Auction House** — the panel appears on the right
3. Click any item name or icon to search it

## Slash commands

- `/ch` — toggle the panel
- `/ch debug` — show saved recipes info

## Installation

Download the latest release and extract the `CraftOrdersHelper` folder into your `World of Warcraft\_retail_\Interface\AddOns\` directory.
